狐麻熊掌｜畢業製作網站

> 一個結合品牌故事、角色設計與環保理念的畢業專題網站  
> 由圖文傳播系畢業生共同開發與設計

---

專案簡介

「狐麻熊掌」是一款以「故事式野餐環保便當盒」為主題的品牌設計專案，透過原創角色與插圖，傳達日系野餐的生活儀式感。本網站為此專題的官方網站，由我們使用 **React.js** 開發，整合圖片展示、留言功能與展出資訊等內容。

- Netlify 靜態網站部署

---

特色功能

- 響應式導覽列與展覽資訊  
- 品牌故事與角色形象設計展示  
- 即時天氣顯示  
- 訪客留言功能  
- Google 地圖嵌入展覽位置
